# Viral Beat — Native App Build Guide

This document explains how to build and publish Viral Beat as a native iOS and Android app using Capacitor.

---

## Prerequisites

| Tool | Version | Notes |
|------|---------|-------|
| Node.js | ≥ 18 | Already installed |
| pnpm | ≥ 8 | Already installed |
| Xcode | ≥ 15 | macOS only, for iOS builds |
| Android Studio | ≥ Hedgehog | For Android builds |
| CocoaPods | ≥ 1.14 | `sudo gem install cocoapods` |
| Java JDK | ≥ 17 | For Android builds |

---

## Quick Start

### 1. Build the web app

```bash
pnpm build
```

This creates `dist/public/` — the web assets Capacitor wraps.

### 2. Sync Capacitor

```bash
npx cap sync
```

This copies `dist/public/` into the native projects and updates plugins.

### 3. Open in IDE

```bash
# iOS (macOS only)
npx cap open ios

# Android
npx cap open android
```

## iOS Build & Publish

### Setup (first time)
```bash
npx cap add ios
cd ios/App && pod install && cd ../..
```

### Build for App Store
1. Open Xcode: `npx cap open ios`
2. Select **Any iOS Device (arm64)** as target
3. Set your **Team** in Signing & Capabilities
4. Set **Bundle Identifier**: `com.viralbeat.app`
5. Increment **Version** and **Build** numbers
6. Product → **Archive**
7. Distribute App → **App Store Connect**

### App Store Requirements
- [ ] App icons (1024×1024 PNG, no alpha)
- [ ] Screenshots for iPhone 6.7", 6.5", 5.5"
- [ ] Privacy policy URL
- [ ] App description (max 4000 chars)

## Android Build & Publish

### Setup (first time)
```bash
npx cap add android
```

### Build for Google Play
1. Open Android Studio: `npx cap open android`
2. Build → **Generate Signed Bundle/APK**
3. Choose **Android App Bundle (.aab)**
4. Create or use existing keystore
5. Upload to Google Play Console

### Google Play Requirements
- [ ] App icons (512×512 PNG)
- [ ] Feature graphic (1024×500 PNG)
- [ ] Screenshots (min 2, max 8 per device type)
- [ ] Privacy policy URL
- [ ] Content rating questionnaire

## Live Reload (Development)

For faster iteration during native development:

1. Find your local IP: `ipconfig getifaddr en0` (macOS)
2. Update `capacitor.config.ts`:
   ```ts
   server: {
     url: 'http://YOUR_IP:3000',
     cleartext: true,
   }
   ```
3. Run: `npx cap sync && npx cap open ios`

## Push Notifications Setup

Viral Beat's ViralMind AI sends proactive trend alerts. To enable:

### iOS
1. Enable **Push Notifications** capability in Xcode
2. Generate APNs key in Apple Developer portal
3. Add key to your notification service

### Android
1. Add `google-services.json` to `android/app/`
2. Enable Firebase Cloud Messaging in Firebase Console
3. Add server key to your notification service

## Troubleshooting

| Issue | Solution |
|-------|----------|
| White screen on launch | Check `webDir` in `capacitor.config.ts` points to `dist/public` |
| API calls failing | Ensure `server.hostname` matches your production domain |
| iOS build fails | Run `pod install` in `ios/App/` directory |
| Android build fails | Sync Gradle: File → Sync Project with Gradle Files |
